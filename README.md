# Parquímetro

Projeto desenvolvido durante o curso de Front-end da EBAC.

## Tecnologias

- HTML5
- CSS3
- JavaScript (POO)

## Funcionalidades

- Seleção do tempo de permanência (30, 60 ou 120 minutos)
- Validação do valor informado
- Cálculo do troco
- Exibição do tempo escolhido e do valor do troco
- Interface responsiva e intuitiva

## Como utilizar

1. Clone o repositório.
2. Abra o arquivo `index.html` em um navegador.
3. Selecione o tempo de permanência desejado.
4. Informe o valor do pagamento.
5. Clique no botão **"Calcular tempo e troco"** para visualizar o resultado.
